Welcome to Assignment Automation.

This project meant to help you speeding and automating the tedious and boring work of preparing homework submission.

That's made using automatic cover creation and pdf merging in the fastest way.

If you are new - follow these steps:

  1. Go to Covers and create a new cover with the appropriate Course Name.
  2. Edit the cover you have just created to match all the information regarding this course's cover.
  3. Go to Prepare Submission, fill the fields, and upload the scan as pdf in the matching field. (don't worry, no data is going to be saved on the server!)
  4. Download the finished submission file.
  5. Good luck!
